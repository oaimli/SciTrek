You may refer to the following introduction of the tables when understanding them.


Table 1 contains meta information of articles, with the following columns:

- article_id: the unique identifier of the article;

- article_title: the title of the article;

- title_word_count: the number of words in the article's title;

- author_count: the number of authors for the article;

- reference_count: the number of references cited in the article.


Table 2 contains author information of the articles, with the following columns:

- article_id: the identifier of the associated article;

- author_name: the name of the author;

- author_position: the position of the author in the author list (starting from 0 for the first author).


Table 3 contains citation relations among the articles, with the following columns:

- article_id_citing: the identifier of the article which cites the other article;

- article_id_cited: the identifier of the article which is cited by the other article.


All the tables can contain multiple entries, while Table 3 may contain no entries; Table 2 maps authors to articles in Table 1, i.e., one author can contribute to multiple articles and one article can have multiple authors; and Table 3 represents citation relationships among articles in Table 1, where one article can be cited by multiple others.
